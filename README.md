# Camp Gaming MOTD

صفحه‌ی معرفی مجموعه‌ی **Camp Gaming** برای نمایش اطلاعات سرورهای Counter-Strike 1.6، قوانین و راه‌های ارتباطی مجموعه.

## امکانات

- معرفی سرورهای Camp Gaming و آدرس اتصال آن‌ها
- امکان کپی سریع آدرس هر سرور
- نمایش شبکه‌های اجتماعی و راه‌های ارتباطی
- ارائه‌ی قوانین مجموعه
- طراحی فارسی، واکنش‌گرا و سازگار با موبایل
- اجرا بدون نیاز به فریم‌ورک یا فرایند Build

## تکنولوژی‌ها

- HTML5
- CSS3
- JavaScript

## اجرای پروژه

پروژه به نصب وابستگی خاصی نیاز ندارد. فایل `index.html` را مستقیماً در مرورگر باز کنید یا پوشه‌ی پروژه را در مسیر وب‌سرور محلی قرار دهید.

برای اجرا با XAMPP می‌توانید این نشانی را باز کنید:

```text
http://localhost/cspage/
```

## ساختار پروژه

```text
CampGamingMOTD/
├── index.html       # صفحه و منطق اصلی
├── hero.png         # تصویر بخش Hero
├── icons/           # آیکن‌های رابط کاربری
└── fonts/           # فونت‌های محلی (در Git نادیده گرفته می‌شوند)
```
